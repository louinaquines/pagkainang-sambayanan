🌟 Project Overview

Pagkainang Sambayanan is a web-based food donation platform designed to bridge the gap between food surplus and community hunger.
    
    The Problem: In urban areas like Cebu and Mandaue City, significant amounts of edible food from restaurants, bakeries, and households are wasted daily, while many vulnerable sectors struggle with food insecurity.
    
    The Solution: This platform provides a real-time connection between Donors (who have surplus food) and Verified Charities (who distribute food to those in need). It uses a role-based system to ensure that donations are tracked, verified, and delivered safely.
    

🌍 Sustainable Development Goals (SDG) Alignment

This project directly contributes to the United Nations' 2030 Agenda:

    SDG 2: Zero Hunger – By facilitating the efficient distribution of surplus food to hungry communities.
    
    SDG 12: Responsible Consumption and Production – By reducing food waste at the retail and consumer levels.
    
    SDG 11: Sustainable Cities and Communities – By building a localized, resilient support system for urban food security.
    
🛠 Tech Stack

This project utilizes a modern Full-Stack architecture focused on scalability, security, and rapid deployment.

    Category                   Technology                           Purpose
    
    Backend                    Laravel 11 (PHP 8.x)                 Core application logic, routing, and API management.
    
    Frontend                   Blade & Tailwind CSS                 Custom "Soft/Pastel" UI/UX design for a clean, user-friendly experience.
    
    Database                   MySQL (via Railway/Supabase)         Relational data storage for users, donations, and charity records.
    
    Authentication             Laravel Socialite                    Secure Google Login integration for quick user onboarding.
    
    Cloud Infrastructure       Railway                              Automated deployment, hosting, and environment management.
    
    Version Control            GitHub                               Repository management with a structured /src and /docs hierarchy.
